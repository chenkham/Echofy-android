## About

**Echofy** is an open-source YouTube Music client for Android. It provides a modern Material Design 3 interface with advanced features for streaming and managing your music.

### Key Features

- **Ad-Free Experience**: Enjoy music without interruptions
- **Background Playback**: Listen while using other apps
- **Offline Mode**: Download songs for offline listening
- **Material Design 3**: Modern and beautiful interface with dynamic colors
- **Lyrics Support**: View synchronized lyrics while listening
- **Audio Normalization**: Consistent volume across all tracks
- **Sleep Timer**: Set a timer to stop playback
- **Custom Playlists**: Create and manage your playlists locally
- **Queue Management**: Full control over your playback queue
- **Discord Rich Presence**: Share what you're listening to

### Requirements

- Android 7.0 (Nougat) or higher

Download the latest APK from the [Releases](https://github.com/Chenkham/Echofy/releases) page.

## Building from Source

### Prerequisites

- Android Studio (latest stable version)
- JDK 21 or higher
- Android SDK with API level 35

### Build Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/Chenkham/Echofy.git
   cd Echofy
   ```

2. Open the project in Android Studio

3. Sync Gradle and build:
   ```bash
   ./gradlew assembleRelease
   ```

4. Find the APK in `app/build/outputs/apk/release/`

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design System**: Material Design 3
- **Architecture**: MVVM with Clean Architecture
- **Dependency Injection**: Hilt
- **Database**: Room
- **Networking**: Ktor Client
- **Media Playback**: Media3 ExoPlayer
